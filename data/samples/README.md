# Sample Data

This directory contains sample datasets for testing and demonstration.

## Available Samples

- **`massive_dirty_dataset.csv`** - Comprehensive test dataset (5,600 rows, 12,321+ issues)
- **`test_data_for_frontend.csv`** - Frontend UI testing sample
- **`test_messy_data.csv`** - Various data quality problems example

## Dataset Descriptions

### Massive Dirty Dataset
Generated by `/scripts/generate_dirty_dataset.py` with realistic data quality issues:
- **Size**: 5,600 rows × 8 columns
- **Missing Values**: 11,721 instances (26% of data)
- **Duplicates**: 599 duplicate records
- **Empty Columns**: 3 completely empty columns
- **Quality Issues**: Name variations, age outliers, salary inconsistencies

### Test Data for Frontend
Small, manageable dataset perfect for:
- UI testing and demonstration
- Quick validation of cleaning algorithms
- Development and debugging

### Test Messy Data
Contains specific examples of common data quality problems:
- Mixed data types
- Inconsistent formatting
- Special characters and encoding issues
- Date format variations

## Usage

These files are automatically used by the test suite and can be uploaded through the web interface for demonstration purposes.

## Adding New Samples

When adding new sample datasets:
1. Place CSV files in this directory
2. Update this README with descriptions
3. Ensure files are under 10MB for web upload compatibility
4. Include variety of realistic data quality issues
